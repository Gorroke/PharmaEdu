# 퀴즈 시스템 개선 작업 노트

이 폴더는 동적 퀴즈 생성기 개선을 위한 분석/계획 노트입니다.
빌드/런타임에 영향 없음. 작업 진행되면 여기 문서들도 업데이트.

## 파일

| 파일 | 내용 |
|---|---|
| `00_analysis.md` | 현재 퀴즈 시스템 구조 분석, "얼토당토없음" 원인 진단 |
| `01_plan.md` | wipe-and-reseed 전략, 단계별 작업 계획, 미결정 사항 |

## 핵심 결정

- **재시드 방식**: 기존 quiz_templates / drug_master 시드는 전부 초기화 후 새로 작성
- **clinical_group 도입**: 약품을 임상 그룹으로 묶어 처방 조합의 임상적 realism 확보
- **신규 question type 활성화**: multi_step / error_spot / fill_blank — 렌더러는 이미 존재, 생성기만 확장하면 됨
