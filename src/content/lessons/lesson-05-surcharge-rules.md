# Lesson 5: 가산 규칙 — 언제, 얼마나 더 받을 수 있나?

> 야간에 약을 조제하면 낮보다 더 받는다. 가루약을 만들면 별도 가산이 붙는다. 8가지 가산의 종류와 '어떤 게 먼저냐'는 우선순위 규칙을 배웁니다.

## 🎯 이 레슨의 목표
- 가산의 8가지 종류와 각각의 적용 조건을 설명할 수 있다
- 가산 우선순위 규칙을 이해하고, 여러 조건이 겹칠 때 어떤 가산이 적용되는지 판단할 수 있다
- 실제 상황을 보고 적용 가산을 스스로 결정할 수 있다

## 📚 시작하기 전에

이 레슨은 **Lesson 4: 조제료 이해하기 (Z코드)**를 먼저 읽고 오시는 것을 권장합니다. 가산은 조제료 위에 얹히는 추가 요금이기 때문에, 조제료 Z코드가 어떤 구조인지 이미 알고 계시면 훨씬 수월하게 이해하실 수 있어요.

---

## 1. 왜 가산이 존재할까? (The Why)

오후 2시에 약국을 찾는 것과 밤 11시에 약국 문을 두드리는 것은 다릅니다. 심야에 문을 여는 약국은 인건비도 더 들고, 약사도 더 힘듭니다. 가루약을 만드는 것은 완제 알약을 그냥 봉투에 담는 것보다 손이 훨씬 많이 갑니다.

**건강보험 제도는 이 차이를 인정합니다.** 일반적인 낮 시간대 조제보다 더 어렵고 부담이 큰 상황에는 기본 조제료에 추가 금액을 붙여줍니다. 이것이 **가산(加算)**, 즉 더하여 계산하는 금액입니다.

가산을 이해하면 약국 영수증에서 "Z2000010"처럼 코드 뒤에 숫자가 붙는 이유도 자연스럽게 알게 됩니다. 그 접미사 숫자들이 바로 어떤 가산인지를 나타내는 신호입니다.

---

## 2. 8가지 가산 한눈에 보기 (The What)

가산의 종류는 총 8가지입니다. 처음 보면 많아 보이지만, 기준별로 묶어서 보면 이해하기 쉽습니다.

| # | 가산 종류 | 적용 조건 | 산정코드 특징 |
|---|---------|---------|------------|
| 1 | **야간 가산** | 평일 오전 9시 이전 또는 오후 6시 이후 / 토요 오후 1시 이후 | 코드 접미사 `010` |
| 2 | **공휴일 가산** | 법정 공휴일 전일 | 코드 접미사 `050` |
| 3 | **토요 오후 가산** | 토요일 오전 9시~오후 1시 (오후 이후는 야간으로 전환) | 코드 접미사 `030`, 별도 행으로 추가 |
| 4 | **6세 미만 가산** | 환자 나이 6세 미만 | 코드 접미사 `600` |
| 5 | **소아 심야 가산** | 6세 미만 + 심야 시간대 (자정~새벽 6시 기준) | 코드 접미사 별도 (`64` 계열) |
| 6 | **가루약(산제) 가산** | 처방에 가루약이 포함된 경우 | 내복약 조제료 코드에 적용 |
| 7 | **비대면 조제 가산** | 비대면 조제(약물안전서비스 U 구분) | 기존 코드 대신 ZC계열 코드로 완전 대체 |
| 8 | **차등 수가** | 약국 영업시간 기반, 야간/심야 시간대 운영 여부 | Z1000(약국관리료)에 반영 |

> 💡 **팁**: 코드 접미사만 봐도 어떤 가산인지 알 수 있어요. `010` = 야간, `050` = 공휴일, `030` = 토요, `600` = 6세 미만. 예를 들어 `Z2000010`은 "조제기본료 + 야간 가산"입니다.

## 3. 각 가산 자세히 살펴보기

### 야간 가산 — "늦게까지 열어줘서 고맙습니다"

야간 가산의 기준 시간은 간단합니다.

- **평일**: 오전 9시 이전, 또는 오후 6시 이후
- **토요일**: 오전 9시 이전, 또는 오후 1시 이후

예를 들어 오후 7시에 처방전을 가지고 약국에 갔다면, 그 약국이 평일에 문을 열고 있다면 야간 가산이 붙습니다. 토요일 오후 2시라면? 오후 1시 넘었으니 역시 야간 가산입니다.

### 공휴일 가산 — "쉬는 날 나와줘서 고맙습니다"

법정 공휴일에는 공휴일 가산이 적용됩니다. 계산 방식은 야간 가산과 동일하지만, 코드 접미사만 `050`으로 달라집니다. 공휴일이 되면 야간 여부는 따지지 않고 공휴일 가산이 우선 적용됩니다.

> ⚠️ **주의**: 야간과 공휴일 가산은 **동시에 적용되지 않습니다.** 공휴일 야간이라도 공휴일 가산 하나만 붙습니다. 둘은 배타적 관계입니다.

### 6세 미만 가산 — "소아 환자는 특별히"

6세 미만 아이에게 조제할 때는 소아 가산이 추가됩니다. 가루약을 먹이는 경우가 많고, 용량 계산도 세심한 주의가 필요하기 때문입니다.

**이 가산의 특별한 점은 다른 가산과 중복 적용이 가능하다는 것입니다.** 야간에 6세 미만 환자가 오면 야간 가산 + 소아 가산을 함께 받을 수 있어요. 코드로 표현하면 `Z2000610` (6세+야간)이 됩니다.

### 가루약 가산 — "갈고 섞는 수고 인정합니다"

처방전에 가루약(산제)이 포함되어 있으면 가루약 가산이 붙습니다. 완성된 알약을 그냥 세어 담는 것과 달리, 가루약은 약사가 직접 분쇄하고 포장해야 하는 수작업이 필요합니다. 2023년 11월 1일부터 새로운 체계가 시행되었습니다.

> ⚠️ **주의**: 가루약 가산은 **내복약 조제료(Z41/Z43 계열)에만 적용**됩니다. 조제기본료(Z2000), 복약지도료(Z3000), 외용약 조제료에는 적용되지 않습니다.

### 비대면 가산 — "직접 오지 않아도 됩니다"

비대면 조제(전화 또는 온라인으로 접수, 배달 또는 대리 수령)의 경우 기존 가산 코드 대신 ZC001~ZC004 코드로 완전히 교체됩니다. 시간대와 공휴 여부에 따라 네 가지 코드 중 하나가 선택됩니다.

## 4. 가장 중요한 것: 우선순위 규칙 (The How)

여러 조건이 동시에 해당될 때 어떤 가산을 적용할지는 **엄격한 우선순위 규칙**으로 결정됩니다. 이 규칙을 잘못 이해하면 청구 금액이 달라지므로 매우 중요합니다.

### 우선순위 요약도

```
조건 확인 시작
     ↓
① 비대면 조제인가? → YES → ZC계열 코드로 교체, 모든 가산 종료
     ↓ NO
② 가루약 처방인가? → YES → 가루약 가산만 적용, 야간/공휴/소아 모두 배제
     ↓ NO
③ 야간 또는 공휴일인가? → YES → 야간 또는 공휴 중 하나만 적용 (배타적)
     ↓ NO
④ 6세 미만 + 심야인가? → YES → 소아 심야 가산 적용
     ↓ NO
⑤ 토요 오전 9시~13시인가? → YES → 토요 가산 적용

[별도] 6세 미만이면 ②를 제외한 ③~⑤와 중복 적용 가능
```

> **정리**: 가루약이 있으면 무조건 가루약 가산만 적용됩니다. 가루약 처방 시에는 야간이든 공휴일이든 소아든 전부 무시됩니다.

### 예시 ①: 평일 오후 7시, 30세 환자, 일반 알약

- 평일 오후 7시 → 야간 시간대 (오후 6시 이후)
- 6세 미만? 아니오 → 소아 가산 해당 없음
- 가루약? 아니오 → 가루약 가산 해당 없음
- 비대면? 아니오

**결론: 야간 가산 1개만 적용.** 조제기본료 코드는 `Z2000010`

### 예시 ②: 토요일 오전 11시, 4세 환자, 일반 알약

- 토요 오전 11시 → 9시~13시 범위이므로 토요 가산 해당
- 4세 → 6세 미만이므로 소아 가산 해당
- 가루약? 아니오 → 가루약 가산 해당 없음

**결론: 토요 가산 + 소아 가산 동시 적용.** 소아는 토요와 중복 가능하기 때문입니다. 조제기본료 코드는 `Z2000630` (6세+토요)

## 6. 지식 체크

**Q1.** 토요일 오후 7시, 만 3세 아이가 가루약 처방을 가지고 약국에 왔습니다. 이때 적용되는 가산은?

① 토요 가산 + 소아 가산  
② 야간 가산 + 소아 가산  
③ 가루약 가산만 적용  
④ 야간 가산 + 가루약 가산

<details>
<summary>정답 확인</summary>

**정답: ③** — 가루약이 있으면 가루약 가산이 최우선입니다. 야간, 소아, 토요 등 다른 모든 가산은 배제됩니다. 비록 토요 오후 7시(야간)이고 3세 소아이더라도, 가루약 가산 하나만 적용됩니다.

</details>

<!-- KNOWLEDGE_CHECK:id=lesson05-check2 -->

**Q2.** 공휴일 오후 10시에 만 8세 환자가 일반 알약 처방전을 가지고 왔습니다. 어떤 가산이 적용될까요?

① 야간 가산만 적용  
② 공휴일 가산만 적용  
③ 야간 가산 + 공휴일 가산 동시 적용  
④ 소아 심야 가산 적용

<details>
<summary>정답 확인</summary>

**정답: ②** — 야간과 공휴일 가산은 배타적입니다. 공휴일이면 시간대와 관계없이 공휴일 가산 하나만 적용됩니다. 8세이므로 6세 미만 소아 가산도 해당 없습니다.

</details>

## 🎓 이 레슨에서 배운 것

- 가산은 야간·공휴·토요·6세 미만·소아심야·가루약·비대면·차등 총 8종류
- 가산 코드는 접미사로 구분: `010`=야간, `050`=공휴, `030`=토요, `600`=6세 미만
- **우선순위**: 비대면 → 가루약 → 야간/공휴(배타적) → 소아심야 → 토요
- 6세 미만(소아) 가산은 예외적으로 야간·공휴·토요와 **중복 적용 가능**
- 가루약 가산이 적용되면 소아 가산을 포함한 다른 모든 가산이 배제됨
